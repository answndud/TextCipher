# 🔐 Text Cipher - 텍스트 암호화/복호화

> 다양한 고전 암호화 기법으로 텍스트를 암호화하고 복호화하세요

## ✨ 주요 기능

### 📜 고전 암호

| 암호 | 설명 | 키 |
|------|------|-----|
| **시저 암호** | 알파벳을 일정 거리만큼 이동 | 시프트 값 (1-25) |
| **ROT13** | 13칸 이동 (암호화=복호화) | 없음 |
| **비즈네르** | 키워드 기반 다중 시프트 | 키워드 |
| **치환 암호** | 알파벳 1:1 매핑 | 키워드 |
| **아트배시** | 알파벳 역순 치환 (A↔Z) | 없음 |
| **역순** | 텍스트 뒤집기 | 없음 |

### 🔐 현대 암호화

| 암호 | 설명 | 키 |
|------|------|-----|
| **AES-256** | 현대 표준 대칭키 암호화 | 비밀번호 (자동 생성 가능) |
| **XOR** | 비트 단위 XOR 연산 | 키 문자열 |

### 📝 인코딩

| 인코딩 | 설명 |
|--------|------|
| **Base64** | 바이너리 → ASCII 변환 |
| **Hex** | 16진수 변환 |
| **Binary** | 2진수 변환 |
| **URL** | URL 안전 인코딩 |
| **Unicode** | \uXXXX 이스케이프 |

### 🔧 난독화

| 방식 | 설명 |
|------|------|
| **모스 부호** | 점과 선으로 변환 |
| **리트스피크** | 1337 스타일 치환 |
| **제로폭 문자** | 보이지 않는 문자로 숨김 |

### 🔍 해시 (단방향)

| 해시 | 설명 |
|------|------|
| **SHA-256** | 256비트 암호학적 해시 |
| **MD5** | 128비트 해시 |

### 🧩 암호 풀기 게임
- 암호화된 텍스트를 해독하는 퍼즐 게임
- 레벨 시스템 및 점수 기록
- 연속 정답 스트릭 추적
- 힌트 시스템

### 📊 빈도 분석
- 문자 출현 빈도 시각화
- 영어 표준 빈도와 비교
- 암호 해독 도구로 활용

## 🎮 사용 방법

### 암호화/복호화

1. **텍스트 입력**: 암호화할 텍스트 입력
2. **암호 방식 선택**: 드롭다운에서 선택
3. **키 설정**: 필요한 경우 키 또는 시프트 값 입력
4. **실행**: "🔒 암호화" 또는 "🔓 복호화" 클릭
5. **복사**: 결과를 클립보드에 복사

### 암호 풀기 게임

1. **게임 탭** 선택
2. 암호화된 텍스트와 힌트 확인
3. 원문 입력 후 "확인" 클릭
4. 레벨업하며 점수 획득!

## 🔐 암호 상세 설명

### 시저 암호 (Caesar Cipher)
가장 유명한 고전 암호. 각 문자를 알파벳에서 고정된 수만큼 이동합니다.

```
평문:  HELLO WORLD
시프트 3:
암호문: KHOOR ZRUOG
```

### ROT13
시저 암호의 특수한 경우로, 13칸 이동합니다. 영어 알파벳이 26자이므로 같은 연산을 두 번 적용하면 원문으로 돌아옵니다.

```
평문:  SECRET
ROT13: FRPERG
ROT13: SECRET (원복)
```

### 비즈네르 암호 (Vigenère Cipher)
키워드를 사용하여 각 문자마다 다른 시프트 값을 적용합니다.

```
평문:  ATTACKATDAWN
키:    LEMONLEMONLE
암호문: LXFOPVEFRNHR
```

### 치환 암호 (Substitution)
알파벳의 각 문자를 다른 문자로 일대일 매핑합니다.

```
원본:   ABCDEFGHIJKLMNOPQRSTUVWXYZ
키워드: KEYWORDABCFGHIJLMNPQSTUVXZ
```

### 아트배시 (Atbash)
히브리어 알파벳에서 유래. 알파벳을 역순으로 치환합니다.

```
A → Z, B → Y, C → X, ...
평문:  HELLO
암호문: SVOOL
```
